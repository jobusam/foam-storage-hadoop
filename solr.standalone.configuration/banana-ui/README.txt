Get the Forensic Data Analysis Website based on Banana:

1) Clone Repository from github
$git clone https://github.com/lucidworks/banana

2) Copy the complete Banana-src folder into solr folder under $SOLR_HOME/server/solr-webapp/webapp
$ cp -R $BANANA_HOME/src $SOLR_HOME/server/solr-webapp/webapp/banana

3) Access the Default Banana UI
$ firefox http://localhost:8983/solr/banana/index.html

4) In the options menu it's possible to load the
configuration file "ForensicData_Init_Configuration_v.0.4_a.json"
-> The configuration files are located in this folder.
